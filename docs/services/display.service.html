Display Service

isLabelIndex(labelIndex, index)

Summary

The isLabelIndex method is a function that checks if a given value index exists in the labelIndex array. It returns true if the value is found and false otherwise.

Example Usage

const display = new DisplayService();

const labelIndex = [1, 2, 3, 4, 5];

const index = 3;

const result = display.isLabelIndex(labelIndex, index);

console.log(result); // Output: true

Code Analysis

Inputs

labelIndex (array): An array of numbers representing the label indices.

index (number): The value to search for in the labelIndex array.

Flow

  1. The isLabelIndex method takes in the labelIndex and index as parameters.
  2. It uses the some method on the labelIndex array to iterate through each element.
  3. Inside the some method, a callback function is executed for each element in the array.
  4. The callback function checks if the current element is equal to index.
  5. If a match is found, the some method returns true and stops further iteration.
  6. If no match is found after iterating through all elements, the some method returns false.

Outputs

  • true if the index value is found in the labelIndex array.
  • false if the index value is not found in the labelIndex array.

findLabelIndex(labelIndex, index)

Summary

The findLabelIndex method is a function that takes in an array labelIndex and a value index, and returns the index of the first occurrence of index in the labelIndex array. If index is not found in the array, it returns -1.

Example Usage

const display = new DisplayService();

const labelIndex = [1, 2, 3, 4, 5];

const index = 3;

const result = display.findLabelIndex(labelIndex, index);

console.log(result); // Output: 2

Code Analysis

  • labelIndex (array): An array of numbers representing the label indices.
  • index (number): The value to search for in the labelIndex array.

Flow

  1. The findLabelIndex method takes in the labelIndex and index as parameters.
  2. It uses the findIndex method on the labelIndex array to find the index of the first occurrence of index.
  3. The findIndex method takes a callback function as an argument, which is executed for each element in the array until a match is found.
  4. Inside the callback function, it checks if the current element is equal to index.
  5. If a match is found, the index of the element is returned.
  6. If no match is found, the findIndex method returns -1.

Outputs

  • The index of the first occurrence of index in the labelIndex array.
  • If index is not found in the array, it returns -1.
