Coding Block CSS Class

The coding-block CSS CLass which wraps around segment of code. The line-number CSS Class is blue color for the line number. I applied this to 01:, 02:, 03:, etc... The comment CSS CLass is for turn the comments green color. I applied this to //This code has been generated by the Console Notebook Command-line Interface. The space-tab CSS Class apply spaces to the left to the code. Applied this to the code let studentsName = ['Bill', 'Tom'];. This CSS Class works as tab. You can apply two tab by wrapping the spans do the follow. For example if you want two tabs add the CSS Classes < span class="space-tab" >< span class="space-tab" >Code< /span >< /span >. As I did for let studentsGrade = [70, 90];
